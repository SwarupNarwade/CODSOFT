So what should i do as a Developer


As a developer, your primary responsibility is to design, code, test, and maintain software applications. The specific tasks you perform may vary depending on your role, the type of development you're involved in, and the industry you work in. Here are some common activities that developers typically engage in:

Requirement Analysis: Understand the project requirements and specifications, often working closely with stakeholders such as clients or product managers.
Design: Create a blueprint for the software, outlining the architecture, data structures, and overall system design.
Coding: Write the actual code for the software based on the design. This involves using programming languages, frameworks, and tools relevant to the project.
Testing: Develop and implement tests to ensure that the software functions correctly and meets the specified requirements. This may include unit testing, integration testing, and system testing.
